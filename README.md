# leaks--v
Приложение для проверки наличия пароля в слитых базах данных с дополнительной функцией для проверки трафика устройства и блокировки подозрительных трекеров


# 1. Клонировать репозиторий
```
git clone https://github.com/yourusername/leaks-shield.git
```
# 2. Открыть в Android Studio
File → Open → выбрать папку проекта

# 3. Синхронизировать Gradle
File → Sync Project with Gradle Files

# 4. Собрать и запустить
Run → Run 'app'


# Запуск unit-тестов
```
./gradlew test
```

# Запуск instrumented-тестов
```
./gradlew connectedAndroidTest
```
